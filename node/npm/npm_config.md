# npm 的配置文件

[https://docs.npmjs.com/misc/config][https://docs.npmjs.com/misc/config]

```bat
rem 获取配置文件
npm config list [-l] [--json]
rem 设置值，会在用户配置文件增加
npm config set <key> <value> [-g|--global]
rem 获取值，返回的是用户配置文件的值，如果没有返回系统默认值
npm config get key
rem 删除值
npm config delete key
```
